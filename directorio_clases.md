# Directorio de Clases - TUP 25 P4

Este directorio contiene archivos y carpetas utilizados para el dictado de clases en el curso TUP 25 P4. A continuación, se detalla cada elemento con una explicación de su contenido y propósito.

## Archivos Individuales

### 03.1.acceso-referencia.js
Archivo JavaScript que explica conceptos fundamentales de JavaScript: constantes, referencias, destructuring, operador rest, y estructuras de datos recursivas. Incluye ejemplos prácticos de manipulación de arrays y objetos.

### 04.1.modulos.js
Explica el concepto de módulos en JavaScript, cómo exportar e importar funcionalidades. Incluye un ejemplo de una agenda simple implementada como módulo.

### 04.2.funciones.js
Introducción a las funciones en JavaScript: definición, parámetros, valores por defecto, argumentos variables, y retorno de múltiples valores. Incluye ejemplos de funciones puras vs impuras.

### 04.3.funciones.js
Continuación del tema de funciones, con ejemplos de uso de `arguments`, parámetros nombrados con destructuring, y funciones flexibles.

### 04.funciones-js.md
Documento Markdown que explica las funciones en JavaScript, con ejemplos de código y explicaciones detalladas.

### 05.1.funciones-lista.js
Funciones para trabajar con listas (arrays): recorridos, mapeo, filtrado, búsqueda, reducción y ordenamiento. Incluye implementaciones manuales y ejemplos de uso.

### 05.1.funciones.js
Más ejemplos de funciones, incluyendo funciones flecha, expresiones lambda, y patrones de orden superior.

### 05.2.funcion-alto-orden.js
Funciones de orden superior: map, filter, reduce. Ejemplos de composición de funciones y algoritmos comunes.

### 05.2.funciones.js
Continuación con funciones flecha, expresiones lambda, y ejemplos de transformación de arrays.

### 05.3.clases.js
Introducción a clases en JavaScript: definición, constructores, métodos, propiedades. Ejemplos con clases Persona y Empleado.

### 05.4.clases.js
Más sobre clases: herencia, getters/setters, métodos estáticos. Ejemplos con clases Persona, Empleado y Gerente.

### 05.4.ordenar.js
Algoritmos de ordenamiento: bubble sort, insertion sort, selection sort. Comparación con el método sort de arrays.

### 05.5.clases.js
Clases con encapsulamiento usando campos privados (#). Ejemplos con clases Persona y Empleado.

### 05.5.funciones-recursiva.js
Funciones recursivas: factorial, búsqueda binaria, algoritmos recursivos para árboles binarios de búsqueda.

### 05.6.clase.js
Clases con herencia y composición. Ejemplos con clases Persona, Empleado y Agenda.

### 05.7.arbol.js
